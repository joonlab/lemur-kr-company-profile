# LEMUR COFFEE & BAKE SHOP — Company Profile

강남 논현동 스페셜티 카페 **LEMUR COFFEE & BAKE SHOP**의 회사 소개서(브랜드 덱).

- 14페이지 슬라이드 덱 · 16:9 (1920×1080)
- 순수 HTML/CSS/JS + `deck-stage` 웹 컴포넌트 — 키보드 **← / → · 스페이스**로 이동, **R** 리셋, 숫자키로 점프
- 브라우저 **Print → PDF 저장**으로 1슬라이드 = 1페이지 PDF 추출

## 보기

- **라이브**: https://lemur-kr-company-profile.vercel.app
- **로컬**: `index.html`을 브라우저로 열기

## 구성

브랜드 스토리 · 대표 바리스타(CHAN) · 비전 · 로스팅 팩토리(스페셜티/시설/품질/산지) · 1→100 수직통합 · 원두 납품(LEMUR BEANS 4종) · 시그니처(RTD·베이크샵) · 카페 컨설팅 · 문의

## 구조

```
index.html            덱 본체 (14 슬라이드)
deck-stage.js         슬라이드 엔진 (웹 컴포넌트)
image-slot.js         이미지 슬롯 컴포넌트
assets/
  ├─ photos/          실제 매장·로스팅·디저트 사진
  ├─ generated/       원두 라인업 이미지 (BEANS 4종)
  └─ brand/           로고 · 여우원숭이 캐릭터 · HACCP 뱃지
```
